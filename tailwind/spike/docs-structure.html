Folder Structure

Spike Bootstrap Dashboard 

              
📦  
│── 📁 dist                                     > Generated production files (final build)  
│── 📁 figma-file                               > Design files for UI reference  
│── 📁 package                                  > Main package directory  
│   │── 📁 demos                                > All demo variations  
│   │   │── 📁 src                              > Source files for demos  
│   │   │   │── 📁 assets                       > CSS, fonts, images, JS, SCSS  
│   │   │   │── 📁 dark                         > Dark mode demo  
│   │   │   │── 📁 docs                         > Documentation files  
│   │   │   │── 📁 horizontal                   > Horizontal layout demo  
│   │   │   │── 📁 landingpage                  > Landing page files  
│   │   │   │── 📁 main                         > Main demo files  
│   │   │   │   │── 📁 partials                 > Header-sidebar components  
│   │   │   │   │── 📄 All HTML files here  
│   │   │   │── 📁 minisidebar                  > Mini sidebar demo  
│   │   │   │── 📁 rtl                          > RTL (Right-to-Left) demo  
│   │   │── 📄 gulpfile.js                      > Gulp automation config  
│   │   │── 📄 .gitignore                       > Ignore unnecessary files for Git  
│   │   │── 📄 package.json                     > Dependencies and npm info  
│   │── 📁 starterkit                           > Starter template for customization  
│   │   │── 📁 src                              > Minimal source files  
│   │   │── 📄 gulpfile.js                      > Gulp settings  
│   │   │── 📄 package.json                     > Dependencies for starterkit  
│── 📄 gulpfile.js                              > Root Gulp configuration  
│── 📄 .gitignore                               > Root Git ignore file  
│── 📄 package.json                             > Root project dependencies  
│── 📄 README.md                                > Theme & project information
