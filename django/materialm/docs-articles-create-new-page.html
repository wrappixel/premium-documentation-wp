    Create a New Django App

    To create a new application inside the MaterialM Django Project, follow the steps given below. In this example, we will create a sample app named testapp.

    First, prepare the folder structure, then register the app, and finally run Django commands to generate and launch it.

    1. Folder Setup

    Below is the recommended folder structure for a new app inside MaterialM. Create this structure after running the app command, or adjust your app to match it. 
    
 materialm/
  config/
|          settings.py
          urls.py
 ...
  static/
        testapp/
               css/
               js/
               images/
  testapp/         your new app folder
       views.py
       urls.py
       models.py
       admin.py
       forms.py
       apps.py
       templates/
               testapp/
                      dashboard.html

    2. Register App in Settings

    Add testapp inside INSTALLED_APPS. 
    
# materialm/settings.py

INSTALLED_APPS = [
    ...
    'testapp',
]

    3. Add URL in Project Routes

    
# materialm/urls.py

from django.urls import path, include

urlpatterns = [
    ...
    path('testapp/', include('testapp.urls')),
]

    4. Create View (Using TemplateHelper)

    MaterialM Django uses TemplateHelper.render for loading pages. Below is the correct example for the testapp app. 
    
# testapp/views.py

from django.shortcuts import render
from webproject.helper.template_helper import TemplateHelper


def dashboard(request):
    context = {
        "page_title": "Test App Dashboard",
    }

    return TemplateHelper.render(
        request=request,
        template_path="testapp/dashboard.html",
        context=context,
        page_title="Test App Dashboard"
    )

    5. Create URLs File

    
# testapp/urls.py

from django.urls import path
from . import views

urlpatterns = [
    path('', views.dashboard, name='testapp_dashboard'),
]

    6. Create Template

    
{% extends "partials/base.html" %}

{% block content %}
<div class="">
  <div class="">
    <h4 class="">Test App Dashboard</h4>
    <p>This is the dashboard page of your new testapp.</p>
  </div>
</div>
{% endblock %}

    7. Create App Command

    Run the command below inside your project directory to generate the app.
    Action Command
    Create testapp
    		 
    
python manage.py startapp testapp
