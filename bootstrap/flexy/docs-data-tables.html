    Datatable

    DataTables is a powerful jQuery plugin that enhances standard HTML tables with advanced interaction features. Built on the principles of progressive enhancement, it offers a highly flexible and customizable solution for adding features such as sorting, pagination, filtering, and more to any HTML table.

    For a complete list of instructions, configuration options, and usage examples, please refer to the official DataTables documentation.

    Usage

    To use DataTables on your page, you must include the required vendor CSS files within the "Vendors CSS" section of the page header.

                  
<link rel="stylesheet" href="@@webRoot/node_modules/datatables.net-bs5/css/dataTables.bootstrap5.min.css" />

    Include the following vendor scripts in the "Vendors JS" section of the page footer to enable DataTables functionality.

                  
<script src="@@webRoot/node_modules/datatables.net/js/jquery.dataTables.min.js"></script>

    Zero Configuration
